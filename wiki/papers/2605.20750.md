# Bijections on the set of extreme points in a compact convex set

**Autori:** Anil Kumar Karn, Susmita Seal · **2026** · [ArXiv](https://arxiv.org/abs/2605.20750) · [PDF](https://arxiv.org/pdf/2605.20750)

## Abstract

Every gauge-reversing bijection on A_c(K), the set of positive real-valued continuous affine functions on a compact convex set K, is completely determined by the induced bijection on the set of extreme points of K. This extends a result by Roelands and Tiersma that A(K) is a JB-algebra if and only if there is a gauge-reversing bijection on A_c(K).
