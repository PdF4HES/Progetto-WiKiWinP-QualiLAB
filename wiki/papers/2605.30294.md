# RAFI -- A Ray/Work Forwarding Infrastructure for Data Parallel Multi-Node/Multi-GPU Computing

**Autori:** Ingo Wald, Serkan Demirci, Alper Sahistan et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.30294) · [PDF](https://arxiv.org/pdf/2605.30294)

## Abstract

We present RaFI, a CUDA and MPI based software framework that simplifies the task of building GPU-enabled data-parallel software where rays or similar work items need to migrate between different GPUs.
