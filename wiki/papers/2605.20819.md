# DynaMate2: Democratization of Agentic AI for Expert-Designed Custom Workflows

**Autori:** Orlando A. Mendible-Barreto et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.20819) · [PDF](https://arxiv.org/pdf/2605.20819)

## Abstract

A hierarchical agentic framework enabling researchers to integrate expert-defined Python functions into AI-callable tools. The LLM routes tasks and selects appropriate tools rather than generating code. DynaMate2 is released as open-source with a web interface, designed as a reusable template for community-driven extension across scientific domains.
