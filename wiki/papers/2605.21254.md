# Random Matrix Spectra from Boltzmann-Weighted Lattice Ensembles

**Autori:** Yaprak Önder, Abbas Ali Saberi, Roderich Moessner · **2026** · [ArXiv](https://arxiv.org/abs/2605.21254) · [PDF](https://arxiv.org/pdf/2605.21254)

## Abstract

A random matrix framework maps equilibrium Boltzmann-sampled lattice configurations to matrix ensembles, with covariance structure inherited from spatial correlations. The framework provides spectral routes to probing collective behaviour in ordered and disordered statistical systems, validated against Monte Carlo data.
