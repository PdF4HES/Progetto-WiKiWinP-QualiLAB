# SpecBench: Evaluating Specification-Level Reasoning for Software Engineering LLM Agents

**Autori:** Grant Hamblin, Kevin Song, Zhanda Zhu et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.30314) · [PDF](https://arxiv.org/pdf/2605.30314)

## Abstract

Software engineering agents are transitioning from code generation to full lifecycle automation. SpecBench evaluates specification-level reasoning: the ability to generate complete, unambiguous, consistent, and correct system specifications. Tasks are derived from the RFC process used by mature open-source projects. For each task, an agent is given an initial design proposal, project codebase, and past RFC discussions, tasked with identifying specification deficiencies. SpecBench contains tasks from 5 diverse repositories: Kubernetes, React, Rust, TVM, and vLLM. The best performing agent, GPT-5.4, achieves 44.4% accuracy.
