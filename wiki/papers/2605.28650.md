# String nets for twisted pivotal categories

**Autori:** Benjamin Haïoun, William Stewart, Filippos Sytilidis · **2026** · [ArXiv](https://arxiv.org/abs/2605.28650) · [PDF](https://arxiv.org/pdf/2605.28650)

## Abstract

Develops graphical calculus for monoidal categories with twisted pivotal structures. The twisted string net modules assemble in an oriented categorified 2-TQFT despite dependence on Morse functions.
