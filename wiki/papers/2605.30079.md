# Intent-Based Orchestration in Open RAN: An ns-3 Simulation Framework

**Autori:** Pouya Agheli, Grégoire Lefebvre · **2026** · [ArXiv](https://arxiv.org/abs/2605.30079) · [PDF](https://arxiv.org/pdf/2605.30079)

## Abstract

This paper presents an extensible ns-3-based simulation framework for evaluating intent-based, semantics-aware control in Open RAN architectures with fine-grained control via internal distributed applications.
