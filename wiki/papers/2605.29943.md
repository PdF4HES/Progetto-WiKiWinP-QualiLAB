# A Domain-Informed Multi-Objective Framework for EEG Channel Selection in Motor Imagery BCIs

**Autori:** Dekka Muni Kumar, Dhruba Jyoti Kalita, Yogesh Kumar Meena · **2026** · [ArXiv](https://arxiv.org/abs/2605.29943) · [PDF](https://arxiv.org/pdf/2605.29943)

## Abstract

This work proposes a multi-objective optimisation framework employing genetic algorithms and particle swarm optimisation for motor imagery classification using EEG signals, balancing spatial relevance and functional discriminability.
