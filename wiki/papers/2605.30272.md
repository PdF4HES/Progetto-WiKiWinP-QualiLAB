# IGA-ODIL: Optimizing DIscrete robust Loss with Isogeometric Analysis to solve forward and inverse problems faster using machine learning tools

**Autori:** Maciej Paszyński, Tomasz Służalec · **2026** · [ArXiv](https://arxiv.org/abs/2605.30272) · [PDF](https://arxiv.org/pdf/2605.30272)

## Abstract

Proposes a spline-based framework combining Optimizing DIscrete Loss (ODIL), robust variational residual minimization, and Isogeometric Analysis for solving PDEs more efficiently than physics-informed neural networks.
