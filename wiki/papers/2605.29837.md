# Weak rank rigidity for groups with a navigable path system

**Autori:** Cornelia Drutu, Davide Spriano, Stefanie Zbinden · **2026** · [ArXiv](https://arxiv.org/abs/2605.29837) · [PDF](https://arxiv.org/pdf/2605.29837)

## Abstract

Demonstrates weak rank rigidity holds for groups with navigable path systems, covering Coxeter groups, hierarchically hyperbolic groups, and others; introduces generalized contraction spaces.
