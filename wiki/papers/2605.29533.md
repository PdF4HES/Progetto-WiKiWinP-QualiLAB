# Uncertainty-triggered wake-up enables energy-efficient, error-resilient edge AI with memristor front ends

**Autori:** Théo Ballet, Aymen Romdhane, Bruno Lovison-Franco et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.29533) · [PDF](https://arxiv.org/pdf/2605.29533)

## Abstract

This work demonstrates that memristor device limitations can be mitigated by coupling a fabricated memristor Bayesian machine to a programmable CPU running a neural network, using uncertainty-triggered wake-up for energy-efficient edge AI.
