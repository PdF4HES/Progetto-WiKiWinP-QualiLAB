# Memory Uncertainty Relation and Harmonic Memory in Random Recurrent Networks

**Autori:** Taichi Haruna, Kohei Nakajima · **2026** · [ArXiv](https://arxiv.org/abs/2605.24628) · [PDF](https://arxiv.org/pdf/2605.24628)

## Abstract

We present an inequality that bounds the short-term memory capability of dynamical systems from below. It can be interpreted as an uncertainty relation between a measure of short-term memory and that of the size of state fluctuations induced by input signals. The lower bound can be achieved by a readout weight and thus represents a suboptimal memory called harmonic memory. We examine analytically and numerically the inequality in a number of reservoir systems subject to input noise, and also study the effect of a state-space regularization. We find that a certain strength of input noise induces extra memory under the regularization, and we refer to this phenomenon as noise-induced memory.
