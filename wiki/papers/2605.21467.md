# DelTA: Discriminative Token Credit Assignment for Reinforcement Learning from Verifiable Rewards

**Autori:** Kaiyi Zhang, Wei Wu, Yankai Lin · **2026** · [ArXiv](https://arxiv.org/abs/2605.21467) · [PDF](https://arxiv.org/pdf/2605.21467)

## Abstract

Introduces a discriminator perspective on RLVR updates for LLM reasoning improvement. Proposes DelTA method to amplify discriminative token-gradient directions, outperforming baselines on mathematical benchmarks and code generation tasks.
