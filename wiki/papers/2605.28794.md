# Non-invertible symmetry enriched string net topological orders

**Autori:** Luisa Eck, Peter Huston, Kyle Kawagoe et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.28794) · [PDF](https://arxiv.org/pdf/2605.28794)

## Abstract

Proposes definitions for non-invertible symmetry enriched topological orders using string net models. All NI-SETOs can be realized in either model, where enriched UFCs can be used to get chiral examples.
