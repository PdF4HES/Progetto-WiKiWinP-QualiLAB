# Mem-π: Adaptive Memory through Learning When and What to Generate

**Autori:** Xiaoqiang Wang et al. · **2026** · [ArXiv](https://arxiv.org/abs/2605.21463) · [PDF](https://arxiv.org/pdf/2605.21463)

## Abstract

Presents adaptive memory framework for LLM agents that generates contextual guidance on-demand rather than retrieving static entries. Uses RL-optimized decision-making to determine when to produce assistance, demonstrating significant gains on web navigation and embodied interaction tasks.
