# Complex spectrum of the partial theta function

**Autori:** Boris Shapiro · **2026** · [ArXiv](https://arxiv.org/abs/2605.29991) · [PDF](https://arxiv.org/pdf/2605.29991)

## Abstract

Investigation of spectral values for the partial theta function where a spectral value is a parameter for which Θ(q,·) has a multiple zero. The analysis combines boundary behavior near the unit circle with interior discrete spectrum studies.
