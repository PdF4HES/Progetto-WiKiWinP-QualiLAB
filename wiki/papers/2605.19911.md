# Reconfigurable Nonlinear Photonic Networks for In-Situ Learning and Memory Formation via Driven-Dissipative Dynamics

**Autori:** Isaac Yorke · **2026** · [ArXiv](https://arxiv.org/abs/2605.19911) · [PDF](https://arxiv.org/pdf/2605.19911)

## Abstract

A Reconfigurable Nonlinear Photonic Decision Network framework is proposed in which computation, memory, and learning emerge directly from driven-dissipative dynamics. Numerical simulations demonstrate simultaneous realization of local physical learning rules enabling adaptive state evolution, tunable stability-plasticity tradeoff, controlled memory formation and erasure via bistable photonic states, fading memory, in-situ learning, and hardware-faithful nonlinear dynamics.
