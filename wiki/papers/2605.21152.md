# The Gompf θ-Invariant of Canonical Contact Structures via Legendrian Surgery

**Autori:** Mohan Bhupal, Burak Ozbagci · **2026** · [ArXiv](https://arxiv.org/abs/2605.21152) · [PDF](https://arxiv.org/pdf/2605.21152)

## Abstract

An explicit Legendrian surgery description of canonical contact structures on oriented links of normal complex surface singularities is provided. Closed-form formulas for Gompf's θ-invariant are derived for Seifert fibered cases using Hirzebruch-Jung continued fractions. Applications include demonstrating that canonical structures minimize θ among diagram-realizable contact structures and ruling out symplectic rational homology ball fillings for certain Stein fillable contact rational homology 3-spheres.
