# Benchmarking Cylindrical Blast Wave Theory Against the OSIRIS-REx Sample Return Capsule Reentry

**Autori:** Elizabeth A. Silber · **2026** · [ArXiv](https://arxiv.org/abs/2605.20455) · [PDF](https://arxiv.org/pdf/2605.20455)

## Abstract

Weak shock theory based on cylindrical blast waves has been used to interpret meteor infrasound but has not been systematically benchmarked against a non-ablating hypersonic source. The OSIRIS-REx Sample Return Capsule reentry on 24 September 2023 provides a benchmark. Using observations from 39 infrasound stations, the study evaluates six blast radius formulations. The Sakurai formulation performs best for non-ablating bodies with 9% linear-period median absolute percentage residuals.
